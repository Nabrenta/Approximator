Аппроксиматор таблично заданных функций
Исполнитель: Абрашина Надежда Александровна

Программа выполняет аппроксимацию (полином лагранжа и полином третьей степени по методу наименших квадратов) таблично заданной функции.
Для изменения функции, перетащите мужный маркер вдоль вертикальной оси или установите его в нужное положение, кликнув на ось.

В папках Solver и TestWork находятся файлы исходного кода программы; TestWork.exe - исполняемый файл; Solver.dll - файл библиотеки, необходимый для работы программы (содержит логику, связанную с построением исходных данных графиков).
